# Citi Bike Data Analysis & Visualization (Tableau)

A Tableau-based data visualization project analyzing Citi Bike usage data to uncover key patterns in ridership, station activity, and user behavior. This project focuses on transforming large-scale transportation data into clear, actionable insights through interactive dashboards and visual storytelling.

---

## Objective
Analyze Citi Bike trip data to identify trends, patterns, and unexpected phenomena in ridership behavior. The goal is to present findings through interactive dashboards and maps designed for non-technical stakeholders, such as city officials and public administrators.

---

## Tools & Technologies
- Tableau Public  
- Data Visualization & Dashboarding  
- Data Cleaning & Transformation  
- Geographic Mapping (Spatial Analysis)  
- Calculated Fields & Aggregations  

---

## Dataset
- Citi Bike Trip History Data (NYC)  
- Selected 1–3 monthly datasets to ensure performance and consistency  
- Includes:
  - Trip duration  
  - Start and end stations  
  - Start and end times  
  - User type (subscriber vs casual)  
  - Geographic location data  

---

## Key Analysis Performed

### Data Preparation
- Cleaned and standardized datasets across multiple time periods  
- Ensured consistent column formatting and data types  
- Removed outliers and invalid data entries  

### Visualization & Dashboard Development
- Created 4–10 visualizations exploring:
  - Ridership trends over time  
  - Peak usage hours  
  - User type distribution  
  - Station popularity  
- Built 2 interactive dashboards, each focused on a distinct data phenomenon  
- Designed a Tableau Story to guide users through insights  

### Geographic Analysis
- Developed a map visualization showing station locations  
- Highlighted station popularity using size and color encoding  
- (Optional) Implemented time-based filtering to show changes over time  

---

## Key Insights
- Identified peak usage times, highlighting commuter behavior patterns  
- Discovered differences in usage between casual riders and subscribers  
- Revealed high-traffic stations concentrated in key urban areas  
- Observed patterns suggesting seasonal or time-based usage trends
